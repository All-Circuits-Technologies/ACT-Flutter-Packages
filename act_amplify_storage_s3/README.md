# Act Amplify Storage <!-- omit from toc -->

This package provides a simple way to interact with the AWS Amplify Storage
service. It implements the `AbstractStorageService` interface from the
`act_storage_manager` package.

## Table of contents <!-- omit from toc -->

- [Features](#features)
- [Documentation](#documentation)

## Features

The features of this packages are the one described in the `act_storage_manager` since
this package is an implementation of the `AbstractStorageService` interface.

In summary, it provides basic method to list, upload, download and delete files from an
AWS simple storage service (S3) bucket.
