## Presentation

This package contains a logger manager for your app

## Config manager usage

| Key                   | Type     | Description                                                      |
| --------------------- | -------- | ---------------------------------------------------------------- |
| `logs.level`          | `string` | Override the LOGS level of logger manager.                       |
| `logs.printInRelease` | `bool`   | Override the printing of LOGS in release for the logger manager. |
